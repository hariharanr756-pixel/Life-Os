# Life OS 🚀

A smart task manager built with HTML, CSS, and JavaScript.

## 📸 Preview
![Life OS](screenshot.png)

## Features
- Add, edit, delete tasks
- Drag & drop reorder
- Dark mode 🌙
- Due dates
- Local storage (data saved)
- Mobile responsive UI
- Smooth animations
